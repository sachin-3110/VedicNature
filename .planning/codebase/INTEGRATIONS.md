# Integrations

## Overview
External services, APIs, databases, and third-party integrations currently present in the codebase.

## Frontend Integrations
- **Google Fonts:** Used for typography (`Noto Serif`, `Manrope`).
- **Google Material Symbols:** Used for UI icons (e.g., `search`, `shopping_bag`, `air`, `play_arrow`, `auto_awesome`, `arrow_forward`).
- **Cloudfront CDN:** Images like the brand logo are being loaded from a Cloudfront URL (e.g., `d1311wbk6unapo.cloudfront.net`).
- **Tailwind CDN:** TailwindCSS loaded directly from `cdn.tailwindcss.com` for rapid styling.

## Backend & APIs
- **None detected.** The project currently exists as a static HTML shell (`design.html`). There is no active API connection, database, auth provider, or webhook integration configured. Forms act as static HTML markup without submittal endpoints.
